# radiance.wiki

Community wiki for [Radiance Cascades](https://radiance.wiki) — a technique for real-time global illumination by Alexander Sannikov.

## Contributing

The easiest way: click **"Edit this page"** on any page at [radiance.wiki](https://radiance.wiki) and propose changes directly on GitHub.

See [CONTRIBUTING.md](CONTRIBUTING.md) for more details.

## Local Development

```bash
git clone https://github.com/OrionReed/radiance-wiki.git
cd radiance-wiki
npm i
npx quartz build --serve
```

Opens at http://localhost:8080.

Built with [Quartz](https://quartz.jzhao.xyz/).
